Customize

There are only a few things you need to change in order to customize the template. If you have access and have used SASS then simply open the corresponding file and make your changes. Otherwise, please follow the instructions below to customize the colors of your choice.

  1. Open an example file to edit, let's open green.css.
  2. The beginning of the file contains everything you can configure: 
    /* This is the background image OR background colour - you may choose. */
.configuration-background {
  background: #81d978;
}

/* This is used as the color of the borders of inputs, buttons, and countdown. */
.configuration-border-color {
  border-color: #053800;
}

/* This is the font color of the page. */
.configuration-font-color {
  color: #053800;
}
  3. Change each of the items to the appropriate colors that you want.
  4. Done! Your changes will now be reflected in green.html



Email management

In order to save and view email addresses that your users have input, you need to do two things

  1. Open your page, click "Find out more" and input your email address, then click the button.
  2. If you get a thank you message, you're good to go! If you get an error like Cannot write to emails.php. Please follow server setup instructions, set the permissions of emails.php to 777 (also known as "CHMOD"). If you are unsure how to do this, please ask your web host.
  3. Open emails.php and update the username and/or password. For example, if you want to set the username to "polly" and the password to "parrot" then the top of the file should look like this: 

    <?php
$username = 'polly';
$password = 'parrot';
  4. Go to www.your-website.com/emails.php and input the credentials when prompted to view all of the emails input by your users. Simple!
